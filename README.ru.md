# Менеджер пользовательского интерфейса
Позволяет игроку сохранять и загружать свою пользовательскую интерфейс на нескольких персонажах.

![Менеджер пользовательского интерфейса](https://i.imgur.com/xkWto2g.png)

## Известные баги
- Если элемент интерфейса не был перемещён игроком, то у него не будет границ UI для сохранения.
